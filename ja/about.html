概要

このサイトの運営者・扱う内容・目的について。

このサイトについて

当サイトは、WPF と .NET による Windows デスクトップ開発の実践的な知見を、2 つの形で公開しています。 具体的な実装課題を掘り下げる技術記事と、その過程で作成したオープンソースのツールです。 WPF で業務アプリケーションやユーティリティを開発する開発者に向けて、概要レベルの解説ではなく実装の参照になる情報を提供することを目的としています。

各記事は、実際に書いて動かしたコードに基づいて執筆しています。 対象フレームワークとバージョン、問題が起きる原因、動作する実装例、エッジケースと落とし穴、代替案とのトレードオフまでを扱います。 公式ドキュメントを言い換えるのではなく、そこでは省かれがちな細部を記録することを方針としています。

扱うトピック

Windows デスクトップ開発と、それを支える C# の言語機能・ライブラリを中心に扱っています。

  • WPF / XAML — 実アプリにおけるコントロール、データバインディング、スタイリング、仮想化、UI 挙動。
  • C# — バージョンごとの言語機能と、日々のコードで効いてくる差異。
  • LINQ と .NET API — 新しい .NET の演算子（.NET 10 まで）を .NET Framework にバックポートする手法を含む。
  • .NET Framework からモダン .NET へ — 移行時の注意点と、ターゲットフレームワーク間の互換性。

運営者について

当サイトは、20 年近くにわたり一貫して Windows デスクトップ開発に携わってきた s-iguchi が制作・執筆しています。 主に .NET と WPF を用い、使いやすさ、シンプルで見やすい UI、そして日々の作業効率を高める実用的なツールの開発を重視しています。

記事の内容は、その開発の現場から直接生まれています。 実際のアプリケーションを作る中で直面した問題、各解決策の背後にある調査、そしてある手法を選んだ理由です。 下記の公開アプリケーションとソースコードは、記事が説明する内容の実践的な裏づけにあたります。
