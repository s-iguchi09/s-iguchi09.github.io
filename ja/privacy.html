プライバシーポリシー

このサイトにおけるデータ・Cookie・広告の取り扱いについて説明します。

最終更新日: 2026年7月17日

概要

このサイト（s-iguchi.github.io）は、技術記事とオープンソースソフトウェアを公開しています。 登録は不要で、訪問者に個人情報の入力を求めることもありません。 本ポリシーでは、サイトの閲覧時に処理されうる限られたデータ（主に広告に関連するもの）について説明します。

サイトが収集するデータ

このサイト自体は個人情報を収集・保存しておらず、独自のアクセス解析やトラッキングも行っていません。 本サイトは GitHub Pages でホスティングされており、一般的な Web ホスティングと同様に、 GitHub がセキュリティおよび運用の目的で標準的なサーバーアクセスログ（IP アドレス、ブラウザーの種類、アクセス日時など）を記録する場合があります。 詳細は GitHub のプライバシーに関する声明をご確認ください。

Cookie について

このサイトは、トラッキングを目的とした独自のファーストパーティ Cookie を設定していません。 閲覧中にブラウザーへ保存されうる Cookie は、下記の第三者による広告配信に由来するものです。 Cookie はブラウザーの設定でいつでも確認・管理・ブロックできます。Cookie をブロックしても、記事の閲覧には支障ありません。

広告（Google AdSense）について

このサイトでは、Google が提供する広告サービス Google AdSense を使用しています。 Google を含む第三者配信事業者は、Cookie に加えて、web ビーコン、IP アドレス、その他の識別子を使用して、ユーザーによる当サイトや他サイトへの過去のアクセス情報に基づいた広告を配信する場合があります。 Google の広告 Cookie の使用により、Google およびそのパートナーは、当サイトやインターネット上の他サイトへのアクセスに基づいて広告を配信できます。

パーソナライズ広告は、以下から管理または無効化できます。

  • Google 広告設定 — Google が表示する広告を管理できます。
  • aboutads.info / Your Online Choices（EU） — 第三者による広告のパーソナライズを無効化できます。

当サイトの利用時に Google がデータをどのように取り扱うかについては、 Google のサービスを使用するサイトやアプリから収集した情報の Google による使用 および Google のプライバシーと利用規約をご確認ください。 EEA・英国・スイスからの訪問者は、上記の設定から広告のパーソナライズを管理できます。

このサイトで紹介するアプリケーションは、外部 API や SDK を利用する場合があります。 その場合は、各サービスのプライバシーポリシーをご確認ください。
