プライバシーポリシー

このサイトにおけるデータの取り扱いについて説明します。

データ収集

このサイトは個人情報を直接収集しません。 ただし、GitHub Pages や解析ツールなどの第三者サービスがアクセスログを収集する場合があります。

第三者サービス

このサイトで紹介するアプリケーションは外部 API や SDK を利用する場合があります。 詳細は各サービスのプライバシーポリシーをご確認ください。

お問い合わせ

プライバシーポリシーに関する質問は GitHub を通じて開発者にお問い合わせください。
