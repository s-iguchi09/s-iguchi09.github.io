MessageBoxResult — デフォルトボタン

MessageBox が表示されたときにフォーカス（デフォルト）を持つボタンを選択するための ComboBox コントロールです。 ユーザーがすぐに Enter キーを押した場合に実行されるボタンを指定します。

コントロール：ComboBox

ComboBox はリストから1つの値を選択できる標準の WPF ドロップダウンコントロールです。 このアプリでは MessageBoxResult フィールドに以下のプロパティを設定した ComboBox を使用しています。

  • IsEditable="False" — 定義済みの列挙値からしか選択できません。
  • ItemsSource="{Binding Source={StaticResource MessageBoxResultDataSource}}" — MessageBoxResult 列挙型のすべての値で自動的に入力されます。
  • SelectedIndex="1" — 起動時に2番目の項目（Cancel）がデフォルト選択されます。
  • SelectedItem="{Binding MessageBoxResult}" — 選択された列挙値が ViewModel にバインドされます。

プロパティ：defaultResult（MessageBoxResult）

defaultResult パラメーター（型：MessageBoxResult）は MessageBox.Show() の第5引数です。 ダイアログ表示時にどのボタンがフォーカスを持つか（デフォルト動作）を指定します。 ユーザーがボタンをクリックせずに Enter キーを押した場合、デフォルトボタンのアクションが実行されます。

重要： 指定したデフォルト結果は、現在の MessageBoxButton 設定に存在するボタンに対応している必要があります。 対応するボタンが存在しない場合（例：MessageBoxButton.OK を設定しているときに Yes を指定するなど）、 ダイアログの最初のボタンがフォーカスを受け取ります。

設定値と動作

None

デフォルトボタンは明示的に指定されません。システムが最初のボタンにフォーカスを設定します。

OK

OK ボタンが初期フォーカスを受け取り、デフォルトアクションになります。 MessageBoxButton が OK または OKCancel のときに有効です。

Cancel

Cancel ボタンが初期フォーカスを受け取り、デフォルトアクションになります。 MessageBoxButton が OKCancel、YesNoCancel、または RetryCancel のときに有効です。

Yes

Yes ボタンが初期フォーカスを受け取り、デフォルトアクションになります。 MessageBoxButton が YesNo または YesNoCancel のときに有効です。

No

No ボタンが初期フォーカスを受け取り、デフォルトアクションになります。 MessageBoxButton が YesNo または YesNoCancel のときに有効です。

コード参照

アプリでは、選択した値が MessageBox.Show() の第5引数として渡されます。 

MessageBox.Show(
    ViewModel.MessageBoxText,
    ViewModel.Caption,
    ViewModel.MessageBoxButton,
    ViewModel.MessageBoxImage,
    ViewModel.MessageBoxResult,  // ← この ComboBox の値（defaultResult として使用）
    ViewModel.MessageBoxOptions
);
