MessageBoxImage — ダイアログアイコン

MessageBox に表示するアイコンを選択するための ComboBox コントロールです。 アイコンによってメッセージの性質（エラー・警告・質問・情報）を視覚的に伝えます。

コントロール：ComboBox

ComboBox はリストから1つの値を選択できる標準の WPF ドロップダウンコントロールです。 このアプリでは MessageBoxImage フィールドに特殊なデータバインディングを使用した ComboBox を使用しています。

  • ItemsSource="{Binding MessageBoxImagePair}" — ViewModel の Dictionary<string, MessageBoxImage> からデータを取得します。これにより、同じアイコンを指す複数の列挙値（Error、Hand、Stop など）を個別の項目として分かりやすい名前で表示できます。
  • DisplayMemberPath="Key" — ドロップダウンに表示されるテキストはディクショナリのキー（フレンドリー名の文字列）です。
  • SelectedValuePath="Value" — バインドされる実際の値はディクショナリの値（MessageBoxImage 列挙型）です。
  • IsEditable="False" — 定義済みのリストからしか選択できません。
  • SelectedIndex="1" — 起動時に2番目の項目（Error）がデフォルト選択されます。

プロパティ：MessageBoxImage

MessageBoxImage 列挙型は MessageBox.Show() の第4引数です。 ダイアログの左側に表示されるアイコンを決定します。

注意： 複数の列挙値が同じアイコンに対応しています。ViewModel ではこれらをディクショナリの個別エントリとして公開し、すべての名前を確認できるようにしています。

設定値と動作

None（値：0）

アイコンは表示されません。ダイアログにはメッセージテキストとボタンのみが表示されます。

Error / Hand / Stop（値：16）

3つの値すべてが同じ白い X が描かれた赤い円のアイコンを表示します。 重大なエラーメッセージや即座の対応が必要な状況に使用します。 ダイアログ表示時にシステムのエラーサウンドが再生されます。

Question（値：32）

白いクエスチョンマーク（?）が描かれた青い円のアイコンを表示します。 ユーザーが決定を下す必要がある確認ダイアログに使用します。

Exclamation / Warning（値：48）

2つの値が同じ黒いエクスクラメーションマーク（!）が描かれた黄色い三角形のアイコンを表示します。 危険な操作についての警告に使用します。 ダイアログ表示時にシステムの警告サウンドが再生されます。

Asterisk / Information（値：64）

2つの値が同じ白い小文字「i」が描かれた青い円のアイコンを表示します。 特定の操作を必要としない情報メッセージに使用します。 ダイアログ表示時にシステムの情報サウンドが再生されます。

コード参照

アプリでは、選択した値が MessageBox.Show() の第4引数として渡されます。 

MessageBox.Show(
    ViewModel.MessageBoxText,
    ViewModel.Caption,
    ViewModel.MessageBoxButton,
    ViewModel.MessageBoxImage,   // ← この ComboBox の値
    ViewModel.MessageBoxResult,
    ViewModel.MessageBoxOptions
);

ViewModel では、表示名と列挙値を対応付けるディクショナリが使用されています。 

public Dictionary<string, MessageBoxImage> MessageBoxImagePair { get; set; } = new()
{
    { "None",        MessageBoxImage.None        },
    { "Error",       MessageBoxImage.Error       },
    { "Hand",        MessageBoxImage.Hand        },
    { "Stop",        MessageBoxImage.Stop        },
    { "Question",    MessageBoxImage.Question    },
    { "Exclamation", MessageBoxImage.Exclamation },
    { "Warning",     MessageBoxImage.Warning     },
    { "Asterisk",    MessageBoxImage.Asterisk    },
    { "Information", MessageBoxImage.Information }
};
