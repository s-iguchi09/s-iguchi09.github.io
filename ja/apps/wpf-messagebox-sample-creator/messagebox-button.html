MessageBoxButton — ボタン構成

MessageBox に表示するボタンの組み合わせを選択するための ComboBox コントロールです。 OK、Cancel、Yes、No、Abort、Retry、Ignore ボタンの表示有無をコントロールします。

← WPF MessageBoxサンプル作成ツールに戻る

コントロール：ComboBox

ComboBox はリストから1つの値を選択できる標準の WPF ドロップダウンコントロールです。 このアプリでは MessageBoxButton フィールドに以下のプロパティを設定した ComboBox を使用しています。

  • IsEditable="False" — 定義済みの値からしか選択できず、カスタムテキストの入力はできません。
  • ItemsSource="{Binding Source={StaticResource MessageBoxButtonDataSource}}" — ObjectDataProvider を使用して MessageBoxButton 列挙型のすべての値で自動的に入力されます。
  • SelectedIndex="1" — アプリ起動時に2番目の項目（OKCancel）がデフォルト選択されます。
  • SelectedItem="{Binding MessageBoxButton}" — 選択された列挙値が ViewModel にバインドされます。

プロパティ：MessageBoxButton

MessageBoxButton 列挙型は MessageBox.Show() の第3引数です。 ダイアログに表示されるボタンの種類を決定します。

設定値と動作

OK

OK ボタンのみを表示します。ユーザーは OK をクリックしてダイアログを閉じます。 単に確認を促す情報メッセージに使用します。 戻り値：MessageBoxResult.OK

OKCancel

OK と Cancel ボタンを表示します。 処理を続行するかキャンセルするかの選択肢を提供する場合に使用します。 戻り値：MessageBoxResult.OK または MessageBoxResult.Cancel

YesNo

Yes と No ボタンを表示します（キャンセルなし）。 保留やキャンセルを許可しない単純な二択決定に使用します。 戻り値：MessageBoxResult.Yes または MessageBoxResult.No

YesNoCancel

Yes、No、Cancel ボタンを表示します。 決定を保留したりキャンセルしたりする第3の選択肢が必要な場合に使用します。 戻り値：MessageBoxResult.Yes、MessageBoxResult.No、または MessageBoxResult.Cancel

AbortRetryIgnore

Abort、Retry、Ignore ボタンを表示します。 エラー回復シナリオで中止・再試行・無視の選択肢を提供する場合に使用します。 戻り値：各ボタンに対応する MessageBoxResult 値

RetryCancel

Retry と Cancel ボタンを表示します。 操作が失敗し、再試行またはキャンセルの選択肢を提供する場合に使用します。 戻り値：クリックされたボタンに対応する MessageBoxResult 値

コード参照

アプリでは、選択した値が MessageBox.Show() の第3引数として渡されます。 

MessageBox.Show(
    ViewModel.MessageBoxText,
    ViewModel.Caption,
    ViewModel.MessageBoxButton,  // ← この ComboBox の値
    ViewModel.MessageBoxImage,
    ViewModel.MessageBoxResult,
    ViewModel.MessageBoxOptions
);
