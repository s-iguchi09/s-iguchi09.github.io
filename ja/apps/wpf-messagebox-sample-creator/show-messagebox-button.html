Show MessageBox — 実行ボタン

設定済みのすべてのパラメーターを使用して MessageBox.Show() を実行する Button コントロールです。このボタンをクリックすると MessageBox ダイアログが表示されます。

← WPF MessageBoxサンプル作成ツールに戻る

コントロール：Button

Button はクリック時にコマンドまたはイベントハンドラーを実行できる標準の WPF インタラクティブコントロールです。 このアプリでは「Show MessageBox.」ボタンに以下の設定が施されています。

  • x:Name="ShowMessageBox" — コードビハインドから参照するための名前を付けています。
  • Grid.ColumnSpan="2" — グリッドレイアウトの両列にまたがり、フォームの全幅を使用します。
  • Click="ShowMessageBox_Click" — コードビハインドの ShowMessageBox_Click イベントハンドラーに接続されています。

フォーム内の他のコントロールと異なり、このボタンは動作にデータバインディングを使用せず、 コードビハインドで ViewModel から直接値を読み取るクラシックなイベントハンドラー方式を採用しています。

動作：MessageBox.Show() の実行

ボタンがクリックされると ShowMessageBox_Click イベントハンドラーが呼び出されます。 ViewModel からすべての値を収集し、MessageBox.Show() を呼び出します。 

private void ShowMessageBox_Click(object sender, RoutedEventArgs e)
{
    if (ViewModel is null)
        return;

    MessageBox.Show(
        ViewModel.MessageBoxText,
        ViewModel.Caption,
        ViewModel.MessageBoxButton,
        ViewModel.MessageBoxImage,
        ViewModel.MessageBoxResult,
        ViewModel.MessageBoxOptions
    );
}

このメソッドはボタンがクリックされた時点で ViewModel から6つの設定値を読み取り、 MessageBox を開きます。ダイアログはモーダルで、ユーザーが閉じるまでアプリケーションはブロックされます。

注意： MessageBox.Show() の戻り値（ユーザーがクリックしたボタンを示す MessageBoxResult）は、 現在のバージョンのアプリでは取得・表示されていません。
