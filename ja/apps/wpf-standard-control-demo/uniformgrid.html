UniformGrid Layout

UniformGrid は子要素のために等しいサイズのセルを自動的に作成するシンプル化された Grid です。すべてのセルが同じ幅と高さを持ち、行と列を手動で定義せずに均等なグリッドを作成するのに最適です。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の UniformGrid デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Columns int グリッドの列数を設定します。0 の場合は自動計算されます。
Rows int グリッドの行数を設定します。0 の場合は自動計算されます。
FirstColumn int 最初の行の最初のセルが始まるオフセットを設定します。
Background (Panel) Brush パネルの背景色を設定します。
ZIndex (Panel attached) int 子要素の描画順序を制御します。
