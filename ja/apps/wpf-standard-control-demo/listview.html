WPF 標準コントロールデモアプリ › ListView

ListView List

ListView は差し替え可能なビューモデルで ListBox を拡張した高度なリストコントロールです。最も一般的なビューは GridView で、サイズ変更や並べ替えをサポートする列ヘッダー付きの表形式でアイテムを表示します。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の ListView デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
SelectionMode (ListBox) Single / Multiple / Extended 選択モードを設定します。
View / Columns (GridView) / AllowsColumnReorder (GridView) ViewBase / GridViewColumnCollection / bool 表示ビューを設定します。GridView の場合は列定義と列の並べ替え許可を設定します。
Width (GridViewColumn) double GridView の各列の幅を設定します。
SelectedIndex / SelectedItem (Selector) int / object 選択されているアイテムのインデックスとオブジェクトを設定します。
IsSelected (ListBoxItem) bool 各アイテムの選択状態を設定します。
HorizontalScrollBarVisibility / VerticalScrollBarVisibility (ScrollViewer) Disabled / Auto / Hidden / Visible スクロールバーの表示設定を制御します。
