TreeView List

TreeView は各アイテムに子アイテムを持つツリー構造で階層データを表示します。アイテムは展開・折りたたみして子アイテムを表示・非表示にできます。データバインドされた階層表示のために HierarchicalDataTemplate をサポートします。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の TreeView デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
IsExpanded (TreeViewItem) bool 各ツリーアイテムが展開されているかどうかを設定します。
IsSelected / IsSelectionActive (TreeViewItem) bool / bool (ReadOnly) アイテムの選択状態と、ツリービューがフォーカスを持っているかどうかを示します。
SelectedValuePath / SelectedValue (TreeView) string / object (ReadOnly) 選択アイテムの特定プロパティ値を SelectedValue として取得します。
ItemTemplate / HierarchicalDataTemplate (ItemsControl) DataTemplate / HierarchicalDataTemplate HierarchicalDataTemplate を使ってデータバインドされた階層構造を表示します。
HorizontalScrollBarVisibility / VerticalScrollBarVisibility (ScrollViewer) Disabled / Auto / Hidden / Visible スクロールバーの表示設定を制御します。
