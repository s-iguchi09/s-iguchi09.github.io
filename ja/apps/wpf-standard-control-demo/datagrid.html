WPF 標準コントロールデモアプリ › DataGrid

DataGrid List

DataGrid はソート、フィルタリング、グループ化、列リサイズ、インプレース編集などの機能を持つ表形式データを表示するための強力なコントロールです。データソースのプロパティから自動的に列を生成するか、カスタム列定義を使用できます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の DataGrid デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
AutoGenerateColumns bool データソースのプロパティから列を自動生成するかどうかを制御します。
IsReadOnly bool グリッド全体を読み取り専用にします。
SelectionMode / SelectionUnit Single/Extended / Cell/FullRow/CellOrRowHeader 選択モードと選択単位（セル/行）を設定します。
ColumnWidth / MaxColumnWidth / MinColumnWidth DataGridLength / double / double 列の幅の設定値、最大値、最小値を設定します。
CanUserAddRows / CanUserDeleteRows bool / bool ユーザーが行を追加・削除できるかどうかを制御します。
CanUserReorderColumns / CanUserResizeColumns / CanUserResizeRows / CanUserSortColumns bool 列の並べ替え・リサイズ、行のリサイズ、列のソートを許可するかどうかを制御します。
RowDetailsVisibilityMode / AreRowDetailsFrozen Collapsed/Visible/VisibleWhenSelected / bool 行詳細パネルの表示モードと、スクロール時に固定するかどうかを設定します。
GridLinesVisibility / HorizontalGridLinesBrush / VerticalGridLinesBrush None/Horizontal/Vertical/All / Brush / Brush グリッド線の表示と色を設定します。
AlternatingRowBackground Brush 交互行の背景色を設定して視認性を高めます。
FrozenColumnCount int 水平スクロール時に固定する列数を設定します。
HeadersVisibility None / Row / Column / All 行ヘッダーと列ヘッダーの表示設定を制御します。
ClipboardCopyMode None / ExcludeHeader / IncludeHeader クリップボードへのコピー時のヘッダー包含設定を制御します。
EnableRowVirtualization / EnableColumnVirtualization bool / bool 行・列の仮想化を有効にしてパフォーマンスを向上させます。
RowValidationErrorTemplate ControlTemplate 行の検証エラーを視覚化するテンプレートを設定します。
RowBackground / RowHeight / MinRowHeight Brush / double / double 行の背景色と高さの設定をします。
RowHeaderWidth / RowHeaderStyle / ColumnHeaderHeight / ColumnHeaderStyle double / Style / double / Style 行ヘッダーと列ヘッダーのサイズとスタイルを設定します。
RowStyle / CellStyle / ColumnHeaderTemplate Style / Style / DataTemplate 行・セル・列ヘッダーのスタイルとテンプレートを設定します。
