InkCanvas Graphics

InkCanvas はユーザーがデジタルインク（スタイラスまたはマウス）で描画できるコントロールです。フリーハンド描画、選択、消去モードをサポートします。ストロークはインクデータとして保存・読み込みができます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の InkCanvas デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
EditingMode None / Ink / GestureOnly / InkAndGesture / Select / EraseByPoint / EraseByStroke 編集モードを設定します。Ink: 描画、Select: 選択、EraseByPoint/Stroke: 消去など。
ActiveEditingMode (ReadOnly) InkCanvasEditingMode 現在アクティブな編集モードを示す読み取り専用プロパティです。
EditingModeInverted InkCanvasEditingMode スタイラスのペン先とは逆の端（消しゴム側）の編集モードを設定します。
DefaultDrawingAttributes DrawingAttributes デフォルトの描画属性（色、太さなど）を設定します。
Strokes StrokeCollection インクキャンバス上のすべてのストロークのコレクションです。
Background Brush InkCanvas の背景色を設定します。
