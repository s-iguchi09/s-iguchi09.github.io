GroupBox Display

GroupBox はヘッダー付きのボーダーコンテナで、関連するコントロールを視覚的にグループ化します。フォーム要素を論理的なセクションに整理するのによく使用されます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の GroupBox デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Header / HasHeader (HeaderedContentControl) object / bool (ReadOnly) グループボックスのヘッダーテキストまたはコンテンツを設定します。HasHeader はヘッダーが設定されているかどうかを示します（読み取り専用）。
HeaderStringFormat (HeaderedContentControl) string ヘッダーが文字列として表示される場合のフォーマット文字列を設定します。
Content (ContentControl) object グループボックス内のコンテンツを設定します。
Padding / FontWeight / FontStyle / FontStretch / FontSize / FontFamily (Control) various パディングとフォント関連のプロパティを設定します。
Background / Foreground (Control) Brush 背景色とテキスト色を設定します。
BorderBrush / BorderThickness (Control) Brush / Thickness ボーダーの色と太さを設定します。
