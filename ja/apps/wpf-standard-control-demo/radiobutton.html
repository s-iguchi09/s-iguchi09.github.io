RadioButton Inputs

RadioButton はグループの中から一つのオプションを選択するコントロールです。同じ GroupName の RadioButton は相互排他的であり、一つを選択すると他は解除されます。ToggleButton を継承しています。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の RadioButton デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
GroupName string (Null = 親コンテナが同じ) 同じ GroupName を持つ RadioButton は互いに排他的になります。GroupName が Null の場合は同一親コンテナ内で排他的になります。
VerticalContentAlignment (Control) Top / Center / Bottom / Stretch ラベルの垂直方向の配置を設定します。
