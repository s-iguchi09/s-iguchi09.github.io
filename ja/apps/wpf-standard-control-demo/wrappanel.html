WrapPanel Layout

WrapPanel は子要素を順番に並べ、スペースが足りなくなると次の行または列に折り返します。利用可能なスペースに応じて折り返すタグリストやツールバーの作成に便利です。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の WrapPanel デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Orientation Horizontal / Vertical 要素を並べる主方向を設定します。Horizontal がデフォルトです。
ItemHeight double Orientation に関係なく、すべての子要素の高さを統一します。未設定の場合は各アイテムのコンテンツ高さが使われます。Horizontal モードで効果が特に分かりやすくなります。
ItemWidth double Orientation に関係なく、すべての子要素の幅を統一します。未設定の場合は各アイテムのコンテンツ幅が使われます。Vertical モードで効果が特に分かりやすくなります。
Background (Panel) Brush パネルの背景色を設定します。
ZIndex (Panel attached) int 子要素の描画順序を制御します。
