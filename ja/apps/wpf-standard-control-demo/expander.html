Expander Display

Expander はユーザーが展開・折りたたみできるコンテナです。常にヘッダーを表示し、トグルボタンをクリックすると内容エリアが表示または非表示になります。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の Expander デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
IsExpanded bool コンテンツが表示されているかどうかを設定します。True で展開、False で折りたたみです。
ExpandDirection Down / Up / Left / Right コンテンツが展開される方向を設定します。
Header / HasHeader (HeaderedContentControl) object / bool (ReadOnly) ヘッダーに表示するテキストまたはコンテンツを設定します。
Content (ContentControl) object 展開時に表示するコンテンツを設定します。
Padding / FontWeight / FontStyle / FontStretch / FontSize / FontFamily (Control) various パディングとフォント関連のプロパティを設定します。
Background / Foreground (Control) Brush 背景色とテキスト色を設定します。
BorderBrush / BorderThickness (Control) Brush / Thickness ボーダーの色と太さを設定します。
