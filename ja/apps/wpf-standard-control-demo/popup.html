Popup Overlays

Popup はメインウィンドウの上にフロートする別ウィンドウにコンテンツを表示するコントロールです。ToolTip と異なり、ホバーで自動表示されず、プログラムによる制御が必要です。多くのオーバーレイコントロールのベースとして使用されます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の Popup デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
IsOpen / StaysOpen / AllowsTransparency bool / bool / bool IsOpen: ポップアップの開閉状態、StaysOpen: ポップアップ外クリックで閉じないかどうか、AllowsTransparency: 透明度サポートの有効化を設定します。
Placement / HorizontalOffset / VerticalOffset PlacementMode / double / double ポップアップの表示位置とオフセットを設定します。
PopupAnimation None / Fade / Slide / Scroll ポップアップが開く際のアニメーションを設定します（AllowsTransparency=True が必要な場合あり）。
Child / PlacementRectangle UIElement / Rect ポップアップ内に表示する要素と、配置基準となる矩形を設定します。
