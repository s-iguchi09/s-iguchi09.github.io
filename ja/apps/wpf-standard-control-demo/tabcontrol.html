TabControl Selectors

TabControl は複数のタブにコンテンツを整理するコンテナです。各タブは TabItem で表されます。ユーザーはタブヘッダーをクリックしてタブを切り替えられます。設定やカテゴリ分けされたコンテンツの整理に最適です。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の TabControl デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
TabStripPlacement Top / Bottom / Left / Right タブヘッダーの位置を設定します。
ContentStringFormat string コンテンツが文字列として表示される場合のフォーマット文字列を設定します。
SelectedContent / SelectedContentStringFormat (ReadOnly) object / string 現在選択されているタブのコンテンツを取得します（読み取り専用）。
