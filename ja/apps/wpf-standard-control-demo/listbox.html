WPF 標準コントロールデモアプリ › ListBox

ListBox List

ListBox はユーザーが一つまたは複数のアイテムを選択できるリストを表示するコントロールです。単一・複数・拡張の選択モードをサポートし、データコレクションにバインドできます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の ListBox デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
SelectionMode Single / Multiple / Extended 選択モードを設定します。Single: 1つのみ、Multiple: 複数、Extended: Ctrl/Shiftキーで複数選択。
DisplayMemberPath (ItemsControl) string アイテムオブジェクトのどのプロパティを表示するかを指定します。
SelectedIndex / SelectedItem (Selector) int / object 選択されているアイテムのインデックスとオブジェクトを取得・設定します。
SelectedValuePath / SelectedValue (Selector) string / object SelectedValuePath で指定したプロパティ値を SelectedValue として取得します。
IsSelected (ListBoxItem) bool 各アイテムが選択されているかどうかを設定します。
HorizontalScrollBarVisibility / VerticalScrollBarVisibility (ScrollViewer) Disabled / Auto / Hidden / Visible スクロールバーの表示設定を制御します。
