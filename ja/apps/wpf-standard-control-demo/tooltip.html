ToolTip Overlays

ToolTip はユーザーが UI 要素にホバーした時に情報を表示する小さなポップアップウィンドウです。プレーンテキストや複雑なコンテンツを格納でき、位置・タイミング・外観の詳細なカスタマイズをサポートします。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の ToolTip デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Placement Mouse / MousePoint / Absolute / AbsolutePoint / Relative / RelativePoint / Bottom / Top / Left / Right / Center / Custom ツールチップの表示位置を設定します。
HorizontalOffset / VerticalOffset double 配置位置からの水平・垂直オフセット量を設定します。
PlacementTarget UIElement ツールチップの位置を基準とする要素を設定します。
PlacementRectangle Rect ツールチップの配置基準となる矩形を設定します。
ShowsToolTipOnKeyboardFocus bool キーボードフォーカス時もツールチップを表示するかどうかを設定します。
HasDropShadow bool ツールチップにドロップシャドウを表示するかどうかを設定します。
InitialShowDelay / ShowDuration / BetweenShowDelay (ToolTipService) int (ms) 初回表示までの遅延時間、表示持続時間、連続表示の間隔を設定します。
Width / MaxWidth / Height / MaxHeight (FrameworkElement) double ツールチップのサイズを制御します。
Background / Foreground / BorderBrush / BorderThickness / Padding (Control) various ツールチップの外観プロパティを設定します。
HorizontalContentAlignment / VerticalContentAlignment (Control) various コンテンツの配置を設定します。
