Menu Menu

Menu はコマンドやオプションの階層リストを表示するコントロールで、通常はアプリウィンドウ上部に水平メニューバーとして表示されます。各アイテムはサブメニューを持てて、キーボードショートカットの表示をサポートします。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の Menu デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
IsMainMenu bool True の場合、Alt キーでメニューが起動し、F10 キーでフォーカスが当たります。
IsCheckable / IsChecked / StaysOpenOnClick (MenuItem) bool / bool / bool MenuItem をチェック可能にし、チェック状態を設定し、クリック後もメニューを開いたままにするかを制御します。
Role (MenuItem) TopLevelItem / TopLevelHeader / SubmenuItem / SubmenuHeader (ReadOnly) MenuItem の役割を示す読み取り専用プロパティです。
Command (MenuItem) ICommand クリック時に実行するコマンドを設定します。
Icon / InputGestureText (MenuItem) object / string メニュー項目の左側に表示するアイコンと、右側に表示するキーボードショートカットのテキストを設定します。
IsHighlighted / IsPressed / IsSubmenuOpen / IsSuspendingPopupAnimation (MenuItem) bool (most ReadOnly) MenuItem の各種状態を示すプロパティ群です（多くは読み取り専用）。
