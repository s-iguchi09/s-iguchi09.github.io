TextBlock Display

TextBlock は少量のテキストを表示するための軽量コントロールです。Label と異なり、フォーカスやキーボードナビゲーションをサポートしません。表示目的に最適化されており、インラインコンテンツ、テキスト書式設定、フロードキュメント要素をサポートします。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の TextBlock デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Text string 表示するテキストを設定します。
TextWrapping NoWrap / Wrap / WrapWithOverflow テキストの折り返し方法を設定します。
TextTrimming None / CharacterEllipsis / WordEllipsis テキストが収まらない場合の省略方法を設定します。
TextAlignment Left / Right / Center / Justify テキストの水平配置を設定します。
Padding Thickness テキストブロック内のパディングを設定します。
FontWeight Thin / ExtraLight / Light / Normal / Medium / SemiBold / Bold / ExtraBold / Black フォントの太さを設定します。
FontStyle Normal / Italic / Oblique フォントのスタイルを設定します。
FontStretch Normal / Condensed / Expanded など フォントの幅の伸縮を設定します。
FontSize double フォントサイズを設定します。
FontFamily string フォントファミリーを設定します。
Background Brush テキストブロックの背景色を設定します。
Foreground Brush テキストの文字色を設定します。
LineHeight / LineStackingStrategy double / BlockLineHeight / MaxHeight 行の高さとスタッキング戦略を設定します。
