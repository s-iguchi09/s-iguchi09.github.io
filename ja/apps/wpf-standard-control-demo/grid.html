WPF 標準コントロールデモアプリ › Grid

Grid Layout

Grid は WPF で最も強力でよく使われるレイアウトパネルです。子要素を行と列に配置し、複雑な構造レイアウトの作成に最適です。比率指定、絶対値、自動サイズの行・列をサポートします。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の Grid デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Column / Row (Grid attached) int 子要素が配置される列・行のインデックスを設定します（0始まり）。
ColumnSpan / RowSpan (Grid attached) int 子要素が占める列数・行数を設定します。
ShowGridLines bool デバッグ用にグリッド線を表示します。
IsSharedSizeScope / SharedSizeGroup (DefinitionBase) bool / string IsSharedSizeScope を持つ Grid 内で、SharedSizeGroup が同じ列/行は同じサイズを共有します。
Width / MinWidth / MaxWidth (ColumnDefinition) GridLength / double / double 列の幅を設定します。*, Auto, または固定値を使用できます。
Height / MinHeight / MaxHeight (RowDefinition) GridLength / double / double 行の高さを設定します。*, Auto, または固定値を使用できます。
Background (Panel) Brush グリッドの背景色を設定します。
ZIndex (Panel attached) int 子要素の描画順序（Z順）を制御します。値が大きいほど手前に描画されます。
