WPF 標準コントロールデモアプリ › DatePicker

DatePicker Inputs

DatePicker はポップアップカレンダーを使ってユーザーが日付を選択できるコントロールです。日付範囲の制限、週の開始曜日の設定、カスタム日付表示形式をサポートします。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の DatePicker デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
SelectedDate DateTime? 現在選択されている日付を取得または設定します。
DisplayDate DateTime カレンダーに表示する月を設定します。
DisplayDateStart / DisplayDateEnd DateTime? 選択可能な日付の範囲の開始と終了を設定します。
FirstDayOfWeek DayOfWeek カレンダーの週の最初の曜日を設定します。
IsDropDownOpen bool カレンダーポップアップが開いているかどうかを制御します。
IsTodayHighlighted bool 今日の日付をカレンダーでハイライト表示するかどうかを制御します。
SelectedDateFormat Short / Long 日付をテキストボックスに表示する際のフォーマットを設定します。
Text string 現在テキストボックスに表示されているテキストを取得します。
