ComboBox Inputs

ComboBox はリストからアイテムを選択できるドロップダウンリストコントロールです。編集可能にして、ユーザーがカスタム値を入力できるようにすることも可能です。シンプルなアイテムリストと複雑なデータバインドコレクションの両方をサポートします。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の ComboBox デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
IsDropDownOpen bool ドロップダウンリストが開いているかどうかを制御します。
IsEditable bool True にするとテキスト入力が可能になります。
IsReadOnly bool IsEditable が True の場合に、テキスト入力を読み取り専用にします。
MaxDropDownHeight double ドロップダウンリストの最大高さを設定します。
Text string IsEditable が True の場合の現在のテキスト値です。
StaysOpenOnEdit bool IsEditable=True の時、テキスト編集中もドロップダウンが開いたままにするかどうかを制御します。
ShouldPreserveUserEnteredPrefix bool オートコンプリート時にユーザーが入力したプレフィックスを保持するかどうかを制御します。
SelectedIndex (Selector) int 選択されているアイテムのインデックス（0始まり）を設定します。
SelectedItem (Selector) object 選択されているアイテムオブジェクトを取得または設定します。
SelectedValue / SelectedValuePath / DisplayMemberPath (ItemsControl) object / string / string SelectedValuePath で指定したプロパティ値を SelectedValue として取得します。DisplayMemberPath で表示プロパティを指定します。
IsSynchronizedWithCurrentItem (ItemsControl) bool? ItemsSource のコレクションビューのカレントアイテムと選択を同期するかどうかを制御します。
