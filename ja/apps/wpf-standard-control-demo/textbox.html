TextBox Inputs

TextBox はユーザーがテキストを入力・編集できる標準入力コントロールです。WPF で最も基本的なコントロールの一つであり、複数行入力、テキスト折り返し、文字種変換など豊富なテキスト編集機能をサポートします。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の TextBox デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Text string バインドまたは直接設定するテキスト値。
TextWrapping NoWrap / Wrap / WrapWithOverflow テキストが領域を超えた場合の折り返し方法を制御します。
TextDecorations None / Underline / Strikethrough / OverLine / Baseline テキストに装飾を追加します（下線など）。
TextAlignment Left / Right / Center / Justify テキストの水平方向の配置を制御します。
MaxLength int (0 = 無制限) 入力可能な最大文字数を設定します。0 は無制限です。
MaxLines int 表示できる最大行数を制限します。
MinLines int 最小表示行数を設定します。
CharacterCasing Normal / Upper / Lower 入力文字を大文字・小文字に自動変換します。
