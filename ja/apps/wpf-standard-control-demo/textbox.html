WPF 標準コントロールデモアプリ › TextBox

TextBox Inputs

TextBox はユーザーがテキストを入力・編集できる最も基本的な入力コントロールです。単一行・複数行モード、テキストの折り返し、読み取り専用モード、テキスト選択のカスタマイズをサポートします。

概要

WPF の TextBox は TextBoxBase を継承したテキスト入力コントロールです。Text プロパティを ViewModel の string プロパティに TwoWay バインドするのが標準的な MVVM パターンです。

AcceptsReturn="True" にすると Enter キーで改行できる複数行テキストボックスになります。TextWrapping="Wrap" と組み合わせると、コンテンツが幅を超えた場合に自動で折り返します。垂直スクロールバーは VerticalScrollBarVisibility="Auto" で追加できます。

リアルタイム入力検証や検索フィルタリングなど、入力のたびに処理が必要な場合は UpdateSourceTrigger="PropertyChanged" を使います。これにより、フォーカスを失ったときではなく文字を入力するたびにバインドソースが更新されます。

IsReadOnly="True" にするとユーザーがテキストを編集できなくなりますが、テキストの選択・コピーは可能な点が TextBlock との大きな違いです。読み取り専用でも選択してコピーできる TextBox は、エラーメッセージや技術情報の表示に便利です。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。

プロパティ設定値説明
Textstringテキストボックスに表示・編集するテキストを設定します。ViewModel の string プロパティに TwoWay バインドします。
AcceptsReturnboolTrue のとき Enter キーで改行できる複数行モードになります。単一行（False/デフォルト）では Enter キーはデフォルトの IsDefault ボタンをアクティブにします。
TextWrappingNoWrap / Wrap / WrapWithOverflowテキストの折り返し方法を設定します。AcceptsReturn=True の複数行モードで Wrap にすると、長い行が自動で折り返されます。
IsReadOnlyboolTrue のとき編集不可になりますが、テキストの選択とコピーは可能です。TextBlock と異なり選択できる点が特徴です。
MaxLengthint入力できる最大文字数を設定します。0（デフォルト）は無制限です。
CaretBrushBrushテキストカーソル（キャレット）の色を設定します。
SelectionBrush / SelectionOpacityBrush / doubleテキスト選択時のハイライト色と不透明度を設定します。
ScrollToEnd() / ScrollToHome() / LineDown()methodsテキストボックスのスクロール位置をプログラムから制御するメソッドです。ログ表示で最新行に自動スクロールする場合に ScrollToEnd() を使用します。
VerticalScrollBarVisibility / HorizontalScrollBarVisibilityDisabled / Auto / Visibleスクロールバーの表示方法を設定します。複数行モードでは VerticalScrollBarVisibility="Auto" が一般的です。

XAML 使用例

<!-- 単一行テキストボックス -->
<TextBox Text="{Binding Name, Mode=TwoWay}"
         MaxLength="100" />

<!-- リアルタイム検索フィルター -->
<TextBox Text="{Binding SearchText, Mode=TwoWay,
               UpdateSourceTrigger=PropertyChanged}"
         PlaceholderText="検索..." />

<!-- 複数行テキストボックス -->
<TextBox AcceptsReturn="True"
         TextWrapping="Wrap"
         VerticalScrollBarVisibility="Auto"
         MinHeight="100"
         Text="{Binding Notes, Mode=TwoWay}"
         VerticalAlignment="Stretch" />

<!-- 読み取り専用でコピー可能 -->
<TextBox IsReadOnly="True"
         Text="{Binding GeneratedCode}"
         FontFamily="Consolas" FontSize="12" />

主な使用例

  • フォームの入力フィールド — 名前・メール・住所など、基本的なテキスト入力フォームに使用します。
  • 検索バー — UpdateSourceTrigger=PropertyChanged を使ってリアルタイムで検索結果をフィルタリングします。
  • メモ・コメント欄 — AcceptsReturn=True の複数行モードで自由記述フィールドを実装します。
  • コード・ログビューアー — IsReadOnly=True と固定幅フォント（Consolas など）で、コードスニペットやログをコピー可能な形式で表示します。
  • 入力バリデーション — Text バインドに ValidationRule や INotifyDataErrorInfo を組み合わせてリアルタイム検証とエラー表示を実装します。

ヒントとベストプラクティス

  • UpdateSourceTrigger は用途に合わせる — リアルタイム検索や検証には PropertyChanged を、送信前の一括検証には LostFocus（デフォルト）を使います。
  • StringFormat でバインド時のフォーマット指定 — 数値を TextBox に表示する場合、StringFormat='{}{0:F2}' でフォーマットを指定できます。
  • Validation.ErrorTemplate をカスタマイズ — バリデーションエラー時の赤枠の代わりにカスタムエラー表示にする場合は Validation.ErrorTemplate を設定します。
  • 複数行モードは AcceptsReturn + TextWrapping — 複数行 TextBox には AcceptsReturn=True と TextWrapping=Wrap を両方設定します。
  • SelectAll() でフォーカス時に全選択 — GotFocus イベントで SelectAll() を呼ぶと、フォーカスした瞬間に既存テキストが全選択されて簡単に上書き入力できます。

関連コントロール

  • TextBlock — 読み取り専用（選択・コピー不可）のテキスト表示。
  • PasswordBox — パスワード（マスク表示）のテキスト入力。
  • ComboBox — IsEditable=True で TextBox 機能を持つドロップダウン。
