WPF 標準コントロールデモアプリ › ToolBar

ToolBar Menu

ToolBar はよく使うコマンドをボタンや他のコントロールとしてコンパクトなストリップに表示するコンテナです。スペースが不足した場合のオーバーフローアイテムをサポートし、複数のツールバーを ToolBarTray に配置できます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の ToolBar デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
HasOverflowItems / IsOverflowItem / IsOverflowOpen / OverflowMode (ToolBar) bool / bool / bool / AsNeeded/Never/Always オーバーフローアイテムの存在確認、アイテムのオーバーフロー状態、オーバーフローメニューの開閉、各アイテムのオーバーフローモードを制御します。
Band / BandIndex (ToolBar) int / int ToolBarTray 内でのツールバーの行（Band）と、その行内での位置（BandIndex）を設定します。
Background / IsLocked / Orientation (ToolBarTray) Brush / bool / Horizontal/Vertical ToolBarTray の背景色、ツールバーの移動をロックするかどうか、配置方向を設定します。
