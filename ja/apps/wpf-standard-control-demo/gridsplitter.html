GridSplitter Resizer

GridSplitter は Grid 内に配置して、ユーザーがドラッグで行や列をリサイズできるコントロールです。リサイズ可能なペインレイアウトの作成によく使用されます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の GridSplitter デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
ResizeDirection Auto / Columns / Rows リサイズする方向（列または行）を設定します。Auto の場合は配置から自動判断します。
ResizeBehavior BasedOnAlignment / CurrentAndNext / PreviousAndCurrent / PreviousAndNext どの行/列のペアをリサイズするかを設定します。
ShowsPreview bool True にするとドラッグ中にプレビューを表示します。False にすると即座にリサイズします。
DragIncrement double ドラッグによるリサイズの最小単位（ピクセル）を設定します。
KeyboardIncrement double キーボード操作によるリサイズの移動量（ピクセル）を設定します。
PreviewStyle Style プレビュー表示のスタイルをカスタマイズします。
IsDragging (Thumb) bool (ReadOnly) 現在ドラッグ中かどうかを示す読み取り専用プロパティです。
