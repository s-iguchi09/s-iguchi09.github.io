ScrollViewer Layout

ScrollViewer はコンテンツが表示領域より大きい場合にスクロール機能を提供するコンテナです。水平・垂直スクロールの両方をサポートし、スクロールバーの表示設定を構成できます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の ScrollViewer デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
VerticalScrollBarVisibility Disabled / Auto / Hidden / Visible 垂直スクロールバーの表示設定を制御します。Auto は必要時のみ表示します。
HorizontalScrollBarVisibility Disabled / Auto / Hidden / Visible 水平スクロールバーの表示設定を制御します。
CanContentScroll bool True の場合、論理スクロール（アイテム単位）を使用します。False の場合は物理スクロール（ピクセル単位）です。
IsDeferredScrollingEnabled bool True の場合、スクロール完了後にコンテンツを更新します（パフォーマンス向上）。
ComputedVerticalScrollBarVisibility / ExtentHeight / ViewportHeight / ScrollableHeight / VerticalOffset / ContentVerticalOffset ReadOnly 垂直スクロールに関する読み取り専用の状態プロパティ群です。
ComputedHorizontalScrollBarVisibility / ExtentWidth / ViewportWidth / ScrollableWidth / HorizontalOffset / ContentHorizontalOffset ReadOnly 水平スクロールに関する読み取り専用の状態プロパティ群です。
