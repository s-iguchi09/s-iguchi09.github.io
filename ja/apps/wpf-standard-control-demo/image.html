WPF 標準コントロールデモアプリ › Image

Image Graphics

Image はファイル、URI、リソースなどさまざまなソースから画像を表示するコントロールです。多様な画像形式をサポートし、コンテナに合わせて画像を引き伸ばす方法を制御するプロパティを提供します。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の Image デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Source ImageSource (URI / file path) 表示する画像のソースを設定します。ファイルパス、URI、またはリソースを指定できます。
Stretch None / Fill / Uniform / UniformToFill 画像をコンテナに合わせるスケーリング方法を設定します。None: そのまま、Fill: 引き伸ばし、Uniform: アスペクト比維持、UniformToFill: 余白なしでアスペクト比維持。
StretchDirection UpOnly / DownOnly / Both スケーリングの方向を制御します。UpOnly: 拡大のみ、DownOnly: 縮小のみ、Both: 両方。

ソースコード

このデモ画面のソースコードは GitHub で公開しています。アプリ内の「コード表示」ボタンから各プロパティの実際の XAML を確認できます。
