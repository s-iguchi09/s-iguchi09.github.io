Viewbox Layout

Viewbox は単一の子要素を利用可能なスペースに合わせてスケールするデコレーターです。アスペクト比を維持しながらコンテンツを自動スケールすることで、解像度に依存しない UI を作成するのに便利です。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の Viewbox デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Stretch / StretchDirection None / Fill / Uniform / UniformToFill / UpOnly / DownOnly / Both Stretch はスケーリング方法、StretchDirection はスケーリングを行う方向（拡大のみ・縮小のみ・両方）を設定します。
