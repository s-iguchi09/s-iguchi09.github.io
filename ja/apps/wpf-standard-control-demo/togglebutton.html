ToggleButton Inputs

ToggleButton はクリック間でチェック/非チェック状態を維持するボタンです。CheckBox と RadioButton の基底クラスです。IsThreeState が有効な場合、IsChecked プロパティは 3 つの状態（null/false/true）を持てます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の ToggleButton デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
IsChecked bool? (Null / False / True) ボタンの現在の状態を設定します。Null は不定（Indeterminate）、False は非チェック、True はチェック済みです。
IsThreeState bool True の場合、IsChecked は Null/False/True の 3 状態をサイクルします。False の場合は False/True の 2 状態のみです。
