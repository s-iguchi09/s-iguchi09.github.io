Button Inputs

Button は WPF で最もよく使われるコマンドトリガーコントロールです。押されると Click イベントが発火し、ICommand インターフェイスを通じたコマンドバインドをサポートします。ButtonBase を継承しており、ClickMode やコマンド関連のプロパティが利用可能です。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の Button デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
IsCancel bool True に設定すると、Escape キー押下でこのボタンの Click が発生します。ダイアログのキャンセルボタンに使用します。
IsDefault bool True に設定すると、Enter キー押下で最初にフォーカスがない場合でもこのボタンの Click が発生します。
ClickMode (ButtonBase) Release / Press / Hover クリックイベントが発生するタイミングを制御します。Release: ボタンを離したとき、Press: 押したとき、Hover: ホバー時。
IsPressed (ButtonBase) bool (ReadOnly) ボタンが現在押されているかどうかを示す読み取り専用プロパティ。
Command (ButtonBase) ICommand ボタンクリック時に実行するコマンドを設定します。ViewModel の ICommand プロパティをバインドします。
CommandParameter (ButtonBase) object コマンド実行時に渡すパラメータを設定します。
