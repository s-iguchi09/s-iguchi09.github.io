CheckBox Inputs

CheckBox はユーザーがオプションを選択・解除できるトグルコントロールです。ToggleButton を継承しており、IsThreeState が有効な場合、チェック済み・非チェック・不定（null）の 3 状態をサポートします。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の CheckBox デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
IsChecked (ToggleButton) bool? (Null / False / True) チェックボックスの状態を設定します。Null は不定状態（Indeterminate）です。
IsThreeState (ToggleButton) bool True にすると 3 段階の状態切り替えが可能になります。不定状態（Null）が追加されます。
VerticalContentAlignment (Control) Top / Center / Bottom / Stretch チェックボックスのコンテンツ（ラベル）の垂直方向の配置を設定します。
