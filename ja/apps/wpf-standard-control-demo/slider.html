Slider Inputs

Slider はトラック上でつまみをドラッグして、連続値または段階値を選択するコントロールです。音量コントロール、ズームレベル、その他の範囲ベースの入力に一般的に使用されます。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の Slider デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Minimum / Maximum / Value (RangeBase) double スライダーの最小値、最大値、現在値を設定します。
Orientation Horizontal / Vertical スライダーの向きを横方向または縦方向に設定します。
SmallChange / LargeChange double SmallChange は矢印キーでの移動量、LargeChange はPage Up/Downでの移動量です。
IsDirectionReversed bool True にすると値の増加方向が逆になります。
TickPlacement None / TopLeft / BottomRight / Both 目盛りの表示位置を設定します。
TickFrequency double 目盛りの間隔を設定します。
IsSnapToTickEnabled bool True にするとつまみが目盛りの位置にスナップします。
Ticks DoubleCollection カスタムな目盛りの位置を指定します。
SelectionStart / SelectionEnd / IsSelectionRangeEnabled double / double / bool 選択範囲（ハイライト領域）の開始値・終了値と有効化を設定します。
AutoToolTipPlacement None / TopLeft / BottomRight ドラッグ中に現在値を表示するツールチップの位置を設定します。
AutoToolTipPrecision int ツールチップに表示する小数点以下の桁数を設定します。
Delay / Interval int (ms) キーボード入力の遅延と繰り返し間隔を設定します（RepeatButton から継承）。
