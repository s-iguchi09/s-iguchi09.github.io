WPF 標準コントロールデモアプリ › Label

Label Display

Label はテキストを表示し、アクセスキー（ニーモニック）をサポートするコンテンツコントロールです。TextBlock と異なり、フォーカスを受け取れ、Target プロパティを使って他のコントロールとペアにしてキーボードナビゲーションに使用するために設計されています。

画面キャプチャ

📷 スクリーンショットは後で追加予定です — アプリ上の Label デモ画面が表示されます。

デモしているプロパティ

以下のプロパティが WPF 標準コントロールデモアプリでインタラクティブにデモされています。 アプリ上でリアルタイムに各プロパティを変更してその動作を確認できます。

プロパティ 設定値 説明
Target UIElement このラベルに関連付けるコントロールを設定します。アクセスキー（Alt+キー）でターゲットにフォーカスが移ります。
Content (ContentControl) object ラベルに表示するコンテンツを設定します。文字列や UIElement が使用可能です。
Padding (Control) Thickness ラベル内のパディングを設定します。
FontWeight / FontStyle / FontStretch / FontSize / FontFamily (Control) various フォント関連のプロパティを設定します。
Background / Foreground (Control) Brush 背景色とテキスト色を設定します。
BorderBrush / BorderThickness (Control) Brush / Thickness ボーダーの色と太さを設定します。
HorizontalContentAlignment / VerticalContentAlignment (Control) Left/Center/Right/Stretch / Top/Center/Bottom/Stretch コンテンツの水平・垂直配置を設定します。
