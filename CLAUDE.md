# CLAUDE.md

このリポジトリで作業する AI エージェント(Claude Code, GitHub Copilot など)向けの共通ガイドです。

## 最初に必ず読むこと

作業を始める前に、**必ず** [`.github/copilot-instructions.md`](.github/copilot-instructions.md) を読み、そこで案内されているルールセットに従ってください。

`.github/copilot-instructions.md` には以下が記載されています:

- リポジトリの概要(日本語・英語の技術ブログサイト)
- 記事の作成・レビュー・翻訳に関するルールセット(`docs/rules/` 以下)の場所
- よく使うプロンプトのクイックガイド

> **Note:** このファイルは [`AGENTS.md`](AGENTS.md) と同じ内容です。Claude Code は `CLAUDE.md` を、その他のエージェントは `AGENTS.md` を起点に読み込みます。
