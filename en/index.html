About the App

This application is designed for learning by allowing you to actually check the behavior of standard controls created with WPF. Developed with the latest .NET 10, it organizes each control into its own dedicated page.

Main Features

  • Check the actual behavior of WPF standard controls
  • Built with the latest .NET 10
  • Each control is separated into its own page for quick access to the information you need
  • Simple UI that helps you stay focused on learning
