Button Inputs

Button is the most commonly used command trigger control in WPF. It fires a Click event when pressed and supports command binding via the ICommand interface. Button inherits from ButtonBase, which provides ClickMode and Command-related properties.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
IsCancel bool When True, pressing Escape triggers this button's Click. Used for dialog cancel buttons.
IsDefault bool When True, pressing Enter triggers this button's Click when no other element handles it.
ClickMode (ButtonBase) Release / Press / Hover Controls when the Click event fires. Release: on mouse up, Press: on mouse down, Hover: on hover.
IsPressed (ButtonBase) bool (ReadOnly) Read-only property indicating whether the button is currently pressed.
Command (ButtonBase) ICommand Sets the command to execute on click. Bind to an ICommand property in the ViewModel.
CommandParameter (ButtonBase) object Sets the parameter to pass when executing the command.
