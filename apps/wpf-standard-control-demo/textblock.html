TextBlock Display

TextBlock is a lightweight control for displaying small amounts of text. Unlike Label, it does not support focus or keyboard navigation. It is optimized for display purposes and supports inline content, text formatting, and flow document elements.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Text string Sets the text to display.
TextWrapping NoWrap / Wrap / WrapWithOverflow Sets how text wraps when it exceeds the available width.
TextTrimming None / CharacterEllipsis / WordEllipsis Sets how text is trimmed when it does not fit.
TextAlignment Left / Right / Center / Justify Sets the horizontal text alignment.
Padding Thickness Sets the padding inside the TextBlock.
FontWeight Thin / ExtraLight / Light / Normal / Medium / SemiBold / Bold / ExtraBold / Black Sets the font weight.
FontStyle Normal / Italic / Oblique Sets the font style.
FontStretch Normal / Condensed / Expanded etc. Sets the font stretch.
FontSize double Sets the font size.
FontFamily string Sets the font family.
Background Brush Sets the background color.
Foreground Brush Sets the text (foreground) color.
LineHeight / LineStackingStrategy double / BlockLineHeight / MaxHeight Sets the line height and line stacking strategy.
