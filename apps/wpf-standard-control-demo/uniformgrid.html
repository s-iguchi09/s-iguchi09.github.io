UniformGrid Layout

UniformGrid is a simplified Grid that automatically creates equally-sized cells for its children. All cells have the same width and height, making it ideal for creating evenly distributed grids without manually defining rows and columns.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Columns int Sets the number of columns. 0 means auto-calculated.
Rows int Sets the number of rows. 0 means auto-calculated.
FirstColumn int Sets the offset for the first cell in the first row.
Background (Panel) Brush Sets the background color of the panel.
ZIndex (Panel attached) int Controls the drawing order of child elements.
