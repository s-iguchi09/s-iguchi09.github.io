RepeatButton Inputs

RepeatButton is a button that continuously fires Click events while it is held down. It is particularly useful for implementing increment/decrement buttons or scroll controls where repeated action is needed without releasing the mouse.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Delay int (ms, default: 250) Delay in milliseconds before repeating starts after the button is first pressed.
Interval int (ms, default: 250) Interval in milliseconds between repeated Click events.
