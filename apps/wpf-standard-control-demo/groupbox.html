GroupBox Display

GroupBox is a bordered container with a header that visually groups related controls. It is commonly used to organize form elements into logical sections.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Header / HasHeader (HeaderedContentControl) object / bool (ReadOnly) Sets the header text or content. HasHeader indicates whether a header is set (read-only).
HeaderStringFormat (HeaderedContentControl) string Sets the format string when the header is displayed as a string.
Content (ContentControl) object Sets the content inside the GroupBox.
Padding / FontWeight / FontStyle / FontStretch / FontSize / FontFamily (Control) various Sets padding and font-related properties.
Background / Foreground (Control) Brush Sets background and foreground (text) colors.
BorderBrush / BorderThickness (Control) Brush / Thickness Sets the border color and thickness.
