Viewbox Layout

Viewbox is a decorator that scales its single child to fill the available space. It is useful for creating resolution-independent UI by automatically scaling content while maintaining aspect ratio.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Stretch / StretchDirection None / Fill / Uniform / UniformToFill / UpOnly / DownOnly / Both Stretch sets the scaling method. StretchDirection sets whether to scale up, down, or both.
