WPF Standard Control Demo App › ListView

ListView List

ListView is an advanced list control that extends ListBox with a pluggable view model. The most common view is GridView, which displays items in a tabular format with column headers that support resizing and reordering.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
SelectionMode (ListBox) Single / Multiple / Extended Sets the selection mode.
View / Columns (GridView) / AllowsColumnReorder (GridView) ViewBase / GridViewColumnCollection / bool Sets the view. For GridView, sets the columns and whether column reordering is allowed.
Width (GridViewColumn) double Sets the width of each GridView column.
SelectedIndex / SelectedItem (Selector) int / object Gets or sets the selected item index and object.
IsSelected (ListBoxItem) bool Sets the selected state of individual items.
HorizontalScrollBarVisibility / VerticalScrollBarVisibility (ScrollViewer) Disabled / Auto / Hidden / Visible Controls the scrollbar visibility.
