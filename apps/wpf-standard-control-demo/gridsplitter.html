GridSplitter Resizer

GridSplitter is a control placed inside a Grid that allows users to resize rows or columns by dragging. It is commonly used to create resizable pane layouts.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
ResizeDirection Auto / Columns / Rows Sets the resize direction (columns or rows). Auto infers from placement.
ResizeBehavior BasedOnAlignment / CurrentAndNext / PreviousAndCurrent / PreviousAndNext Sets which pair of rows/columns to resize.
ShowsPreview bool When True, shows a preview while dragging. When False, resizes immediately.
DragIncrement double Sets the minimum resize increment (pixels) when dragging.
KeyboardIncrement double Sets the resize increment (pixels) for keyboard operations.
PreviewStyle Style Customizes the style of the drag preview.
IsDragging (Thumb) bool (ReadOnly) Read-only property indicating whether dragging is in progress.
