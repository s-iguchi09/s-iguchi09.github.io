ToggleButton Inputs

ToggleButton is a button that maintains its checked/unchecked state between clicks. It is the base class for CheckBox and RadioButton. The IsChecked property can have three states (null/false/true) when IsThreeState is enabled.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
IsChecked bool? (Null / False / True) Sets the current state. Null is indeterminate, False is unchecked, True is checked.
IsThreeState bool When True, IsChecked cycles through Null/False/True. When False, only False/True are toggled.
