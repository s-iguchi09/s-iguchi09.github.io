TabControl Selectors

TabControl is a container that organizes content into multiple tabs. Each tab is represented by a TabItem. Users can switch between tabs by clicking on the tab headers. It is ideal for organizing settings or categorized content.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
TabStripPlacement Top / Bottom / Left / Right Sets the position of the tab headers.
ContentStringFormat string Sets the format string when content is displayed as a string.
SelectedContent / SelectedContentStringFormat (ReadOnly) object / string Gets the content of the currently selected tab (read-only).
