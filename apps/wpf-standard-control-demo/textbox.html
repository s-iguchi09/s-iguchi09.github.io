TextBox Inputs

TextBox is a standard input control that allows users to enter and edit text. It is one of the most fundamental controls in WPF and supports rich text editing features including multiline input, text wrapping, and character casing transformation.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Text string The text value to bind or set directly.
TextWrapping NoWrap / Wrap / WrapWithOverflow Controls how text wraps when it exceeds the available width.
TextDecorations None / Underline / Strikethrough / OverLine / Baseline Adds decorations such as underline to the text.
TextAlignment Left / Right / Center / Justify Controls the horizontal alignment of the text.
MaxLength int (0 = unlimited) Sets the maximum number of characters. 0 means unlimited.
MaxLines int Limits the maximum number of visible lines.
MinLines int Sets the minimum number of visible lines.
CharacterCasing Normal / Upper / Lower Automatically converts input characters to upper or lower case.
