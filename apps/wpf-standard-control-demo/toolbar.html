ToolBar Menu

ToolBar is a container for displaying frequently used commands as buttons or other controls in a compact strip. It supports overflow items when there is not enough space, and multiple toolbars can be arranged in a ToolBarTray.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
HasOverflowItems / IsOverflowItem / IsOverflowOpen / OverflowMode (ToolBar) bool / bool / bool / AsNeeded/Never/Always Checks overflow item existence, item overflow state, overflow menu open state, and each item's overflow mode.
Band / BandIndex (ToolBar) int / int Sets the row (Band) and position within that row (BandIndex) in a ToolBarTray.
Background / IsLocked / Orientation (ToolBarTray) Brush / bool / Horizontal/Vertical Sets ToolBarTray background, locks toolbar movement, and sets layout orientation.
