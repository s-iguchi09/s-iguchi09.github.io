ScrollViewer Layout

ScrollViewer is a container that provides scrolling functionality for its content when the content is larger than the available display area. It supports both horizontal and vertical scrolling with configurable scroll bar visibility.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
VerticalScrollBarVisibility Disabled / Auto / Hidden / Visible Controls the vertical scrollbar visibility. Auto shows it only when needed.
HorizontalScrollBarVisibility Disabled / Auto / Hidden / Visible Controls the horizontal scrollbar visibility.
CanContentScroll bool When True, uses logical scrolling (item-by-item). When False, uses physical scrolling (pixel-by-pixel).
IsDeferredScrollingEnabled bool When True, updates content after scrolling completes (improves performance).
ComputedVerticalScrollBarVisibility / ExtentHeight / ViewportHeight / ScrollableHeight / VerticalOffset / ContentVerticalOffset ReadOnly Read-only state properties related to vertical scrolling.
ComputedHorizontalScrollBarVisibility / ExtentWidth / ViewportWidth / ScrollableWidth / HorizontalOffset / ContentHorizontalOffset ReadOnly Read-only state properties related to horizontal scrolling.
