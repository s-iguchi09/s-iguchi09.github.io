DataGrid List

DataGrid is a powerful control for displaying tabular data with features like sorting, filtering, grouping, column resizing, and in-place editing. It automatically generates columns from data source properties or allows custom column definitions.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
AutoGenerateColumns bool Controls whether columns are auto-generated from data source properties.
IsReadOnly bool Makes the entire grid read-only.
SelectionMode / SelectionUnit Single/Extended / Cell/FullRow/CellOrRowHeader Sets the selection mode and unit (cell or row).
ColumnWidth / MaxColumnWidth / MinColumnWidth DataGridLength / double / double Sets default, maximum, and minimum column widths.
CanUserAddRows / CanUserDeleteRows bool / bool Controls whether users can add or delete rows.
CanUserReorderColumns / CanUserResizeColumns / CanUserResizeRows / CanUserSortColumns bool Controls whether users can reorder/resize columns, resize rows, and sort columns.
RowDetailsVisibilityMode / AreRowDetailsFrozen Collapsed/Visible/VisibleWhenSelected / bool Sets row details panel visibility mode and whether it freezes during scroll.
GridLinesVisibility / HorizontalGridLinesBrush / VerticalGridLinesBrush None/Horizontal/Vertical/All / Brush / Brush Sets grid line visibility and colors.
AlternatingRowBackground Brush Sets the background color of alternating rows to improve readability.
FrozenColumnCount int Sets the number of columns frozen during horizontal scrolling.
HeadersVisibility None / Row / Column / All Controls the visibility of row and column headers.
ClipboardCopyMode None / ExcludeHeader / IncludeHeader Controls whether headers are included when copying to clipboard.
EnableRowVirtualization / EnableColumnVirtualization bool / bool Enables row/column virtualization for improved performance.
RowValidationErrorTemplate ControlTemplate Sets the template to visualize row validation errors.
RowBackground / RowHeight / MinRowHeight Brush / double / double Sets row background color and height values.
RowHeaderWidth / RowHeaderStyle / ColumnHeaderHeight / ColumnHeaderStyle double / Style / double / Style Sets the size and style of row and column headers.
RowStyle / CellStyle / ColumnHeaderTemplate Style / Style / DataTemplate Sets styles and templates for rows, cells, and column headers.
