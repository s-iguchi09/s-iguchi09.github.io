InkCanvas Graphics

InkCanvas is a control that allows users to draw with digital ink (stylus or mouse). It supports freehand drawing, selection, and erasing modes. The strokes can be saved and loaded as ink data.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
EditingMode None / Ink / GestureOnly / InkAndGesture / Select / EraseByPoint / EraseByStroke Sets the editing mode. Ink: draw, Select: select, EraseByPoint/Stroke: erase, etc.
ActiveEditingMode (ReadOnly) InkCanvasEditingMode Read-only property indicating the currently active editing mode.
EditingModeInverted InkCanvasEditingMode Sets the editing mode for the inverted end of the stylus (eraser).
DefaultDrawingAttributes DrawingAttributes Sets the default drawing attributes such as color and size.
Strokes StrokeCollection The collection of all strokes on the InkCanvas.
Background Brush Sets the background color of the InkCanvas.
