Menu Menu

Menu is a control that displays a hierarchical list of commands and options, typically displayed as a horizontal menu bar at the top of an application window. Each item can have submenus and supports keyboard shortcut display.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
IsMainMenu bool When True, the menu is activated with Alt key and receives focus with F10.
IsCheckable / IsChecked / StaysOpenOnClick (MenuItem) bool / bool / bool Makes a MenuItem checkable, sets its checked state, and keeps the menu open after click.
Role (MenuItem) TopLevelItem / TopLevelHeader / SubmenuItem / SubmenuHeader (ReadOnly) Read-only property indicating the role of the MenuItem in the menu hierarchy.
Command (MenuItem) ICommand Sets the command to execute when clicked.
Icon / InputGestureText (MenuItem) object / string Sets the icon displayed on the left and the keyboard shortcut text displayed on the right.
IsHighlighted / IsPressed / IsSubmenuOpen / IsSuspendingPopupAnimation (MenuItem) bool (most ReadOnly) Properties indicating various states of the MenuItem (most are read-only).
