WPF Standard Control Demo App › Label

Label Display

Label is a content control that displays text and supports access keys (mnemonics). Unlike TextBlock, it can receive focus and is designed to be paired with other controls using the Target property for keyboard navigation.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Target UIElement Sets the control associated with this label. The access key (Alt+key) moves focus to the target.
Content (ContentControl) object Sets the content to display. Can be a string or UIElement.
Padding (Control) Thickness Sets the padding inside the label.
FontWeight / FontStyle / FontStretch / FontSize / FontFamily (Control) various Sets font-related properties.
Background / Foreground (Control) Brush Sets the background and foreground (text) colors.
BorderBrush / BorderThickness (Control) Brush / Thickness Sets the border color and thickness.
HorizontalContentAlignment / VerticalContentAlignment (Control) Left/Center/Right/Stretch / Top/Center/Bottom/Stretch Sets the horizontal and vertical alignment of the content.
