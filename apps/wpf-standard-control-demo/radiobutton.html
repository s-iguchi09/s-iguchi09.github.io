RadioButton Inputs

RadioButton allows users to select one option from a group. RadioButtons in the same GroupName are mutually exclusive — selecting one deselects the others. It inherits from ToggleButton.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
GroupName string (Null = same parent container) RadioButtons with the same GroupName are mutually exclusive. When null, exclusivity applies within the same parent container.
VerticalContentAlignment (Control) Top / Center / Bottom / Stretch Sets the vertical alignment of the label content.
