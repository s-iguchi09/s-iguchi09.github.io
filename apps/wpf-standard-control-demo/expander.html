WPF Standard Control Demo App › Expander

Expander Display

Expander is a container that can be expanded and collapsed by the user. It shows a header at all times, and the content area is revealed or hidden when the toggle button is clicked.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
IsExpanded bool Sets whether the content is visible. True expands, False collapses.
ExpandDirection Down / Up / Left / Right Sets the direction in which the content expands.
Header / HasHeader (HeaderedContentControl) object / bool (ReadOnly) Sets the text or content to display in the header.
Content (ContentControl) object Sets the content to display when expanded.
Padding / FontWeight / FontStyle / FontStretch / FontSize / FontFamily (Control) various Sets padding and font-related properties.
Background / Foreground (Control) Brush Sets background and foreground (text) colors.
BorderBrush / BorderThickness (Control) Brush / Thickness Sets the border color and thickness.
