DatePicker Inputs

DatePicker is a control that allows users to select a date using a popup calendar. It supports date range restrictions, first day of week configuration, and custom date display formats.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
SelectedDate DateTime? Gets or sets the currently selected date.
DisplayDate DateTime Sets the month displayed in the calendar.
DisplayDateStart / DisplayDateEnd DateTime? Sets the start and end of the selectable date range.
FirstDayOfWeek DayOfWeek Sets the first day of the week displayed in the calendar.
IsDropDownOpen bool Controls whether the calendar popup is open.
IsTodayHighlighted bool Controls whether today's date is highlighted in the calendar.
SelectedDateFormat Short / Long Sets the format for displaying the date in the text box.
Text string Gets the text currently displayed in the text box.
