Slider Inputs

Slider is a control that allows users to select a value from a continuous or stepped range by dragging a thumb along a track. It is commonly used for volume controls, zoom levels, and other range-based inputs.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Minimum / Maximum / Value (RangeBase) double Sets the minimum, maximum, and current values of the slider.
Orientation Horizontal / Vertical Sets the slider orientation to horizontal or vertical.
SmallChange / LargeChange double SmallChange is the step for arrow keys; LargeChange is the step for Page Up/Down.
IsDirectionReversed bool When True, reverses the direction in which the value increases.
TickPlacement None / TopLeft / BottomRight / Both Sets the position of tick marks.
TickFrequency double Sets the interval between tick marks.
IsSnapToTickEnabled bool When True, the thumb snaps to tick mark positions.
Ticks DoubleCollection Specifies custom tick mark positions.
SelectionStart / SelectionEnd / IsSelectionRangeEnabled double / double / bool Sets the start and end of the selection range highlight, and enables it.
AutoToolTipPlacement None / TopLeft / BottomRight Sets the position of the tooltip showing the current value while dragging.
AutoToolTipPrecision int Sets the number of decimal places shown in the auto-tooltip.
Delay / Interval int (ms) Sets the delay and repeat interval for keyboard input (inherited from RepeatButton).
