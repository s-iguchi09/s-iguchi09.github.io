Grid Layout

Grid is the most powerful and commonly used layout panel in WPF. It arranges child elements in rows and columns, making it ideal for creating complex, structured layouts. It supports proportional, absolute, and auto-sized rows and columns.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Column / Row (Grid attached) int Sets the column and row index where a child element is placed (zero-based).
ColumnSpan / RowSpan (Grid attached) int Sets the number of columns or rows a child element spans.
ShowGridLines bool Shows grid lines for debugging purposes.
IsSharedSizeScope / SharedSizeGroup (DefinitionBase) bool / string Within a Grid with IsSharedSizeScope, columns/rows with the same SharedSizeGroup share their size.
Width / MinWidth / MaxWidth (ColumnDefinition) GridLength / double / double Sets the column width. Can use *, Auto, or a fixed value.
Height / MinHeight / MaxHeight (RowDefinition) GridLength / double / double Sets the row height. Can use *, Auto, or a fixed value.
Background (Panel) Brush Sets the background color of the grid.
ZIndex (Panel attached) int Controls the drawing order (Z-order) of child elements. Higher values render in front.
