ToolTip Overlays

ToolTip is a small popup window that displays information about a UI element when the user hovers over it. It can contain plain text or complex content, and supports extensive customization for positioning, timing, and appearance.

Demonstrated Properties

The following properties are demonstrated interactively in the WPF Standard Control Demo App. Each property can be configured in real time within the app to observe its behavior.

Property Values Description
Placement Mouse / MousePoint / Absolute / AbsolutePoint / Relative / RelativePoint / Bottom / Top / Left / Right / Center / Custom Sets the placement position of the tooltip.
HorizontalOffset / VerticalOffset double Sets the horizontal and vertical offset from the placement position.
PlacementTarget UIElement Sets the element relative to which the tooltip is positioned.
PlacementRectangle Rect Sets the rectangle relative to which the tooltip is positioned.
ShowsToolTipOnKeyboardFocus bool Sets whether the tooltip is shown when the element receives keyboard focus.
HasDropShadow bool Sets whether the tooltip has a drop shadow.
InitialShowDelay / ShowDuration / BetweenShowDelay (ToolTipService) int (ms) Sets the delay before first showing, how long it stays visible, and the delay between consecutive tooltips.
Width / MaxWidth / Height / MaxHeight (FrameworkElement) double Controls the size of the tooltip.
Background / Foreground / BorderBrush / BorderThickness / Padding (Control) various Sets the visual appearance properties of the tooltip.
HorizontalContentAlignment / VerticalContentAlignment (Control) various Sets the content alignment.
