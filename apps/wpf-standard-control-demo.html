WPF Standard Control Demo (.NET 10)

A demonstration of standard WPF controls running on .NET 10.

Overview

This application showcases standard WPF controls using .NET 10. It is intended as a reference for UI behavior, layout, and styling.

Features

  • Demonstrates common WPF controls
  • Modern .NET 10 runtime
  • Useful for UI testing and comparison
