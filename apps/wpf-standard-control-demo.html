WPF Standard Control Demo (.NET 10)

This application demonstrates how to use standard WPF controls in a modern .NET 10 environment. It serves as a practical reference for developers who want to understand layout behavior, styling, and control interactions in real-world scenarios.

Features

  • Demonstrates common WPF controls
  • Shows layout patterns and best practices
  • Includes sample interactions and UI behaviors
