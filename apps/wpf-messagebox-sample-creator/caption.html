Caption — Title Bar Text Input

A single-line TextBox control for entering the title bar text of the MessageBox. This text is displayed in the title bar of the dialog window.

← Back to WPF MessageBox Sample Creator

Control: TextBox

The TextBox is a standard WPF input control that accepts text from the user. In this app, the Caption field uses a single-line TextBox configured with the following properties:

  • HorizontalAlignment="Stretch" — The control stretches to fill the available width.
  • VerticalAlignment="Center" — The control is vertically centered within its row.

The Text property is bound to the Caption property in the ViewModel using UpdateSourceTrigger=PropertyChanged, so the binding updates on every keystroke. Unlike the MessageBoxText field, this TextBox does not have AcceptsReturn="True", meaning it is a single-line input.

Property: caption

The caption parameter is the second argument of MessageBox.Show(). It sets the text displayed in the title bar of the dialog window.

Behavior by Input

Input Behavior
Empty string The title bar of the MessageBox is blank (no text shown).
Short text The text appears as-is in the title bar of the dialog.
Long text The title bar text is truncated with "..." if it exceeds the window width.

Code Reference

In the app, the value from this TextBox is passed as the second argument to MessageBox.Show(): 

MessageBox.Show(
    ViewModel.MessageBoxText,
    ViewModel.Caption,          // ← This TextBox value
    ViewModel.MessageBoxButton,
    ViewModel.MessageBoxImage,
    ViewModel.MessageBoxResult,
    ViewModel.MessageBoxOptions
);
