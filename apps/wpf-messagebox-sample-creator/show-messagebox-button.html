Show MessageBox — Execute Button

A Button control that triggers MessageBox.Show() with all the currently configured parameters. Clicking this button displays the MessageBox dialog.

← Back to WPF MessageBox Sample Creator

Control: Button

The Button is a standard WPF interactive control that executes a command or event handler when clicked. In this app, the "Show MessageBox." button is configured as follows:

  • x:Name="ShowMessageBox" — Named for reference in the code-behind.
  • Grid.ColumnSpan="2" — Spans both columns of the grid layout, filling the full width of the form.
  • Click="ShowMessageBox_Click" — Connected to the ShowMessageBox_Click event handler in the code-behind.

Unlike the other controls in the form, the Button does not use data binding for its behavior — it uses a classic event handler approach, directly reading from the ViewModel in the code-behind.

Behavior: Executing MessageBox.Show()

When the button is clicked, the ShowMessageBox_Click event handler is called. It collects all values from the ViewModel and calls MessageBox.Show() with them: 

private void ShowMessageBox_Click(object sender, RoutedEventArgs e)
{
    if (ViewModel is null)
        return;

    MessageBox.Show(
        ViewModel.MessageBoxText,
        ViewModel.Caption,
        ViewModel.MessageBoxButton,
        ViewModel.MessageBoxImage,
        ViewModel.MessageBoxResult,
        ViewModel.MessageBoxOptions
    );
}

The method reads the six configured values from the ViewModel at the moment the button is clicked and opens the MessageBox. The dialog is modal — the application is blocked until the user closes it.

Note: The return value of MessageBox.Show() (a MessageBoxResult indicating which button the user clicked) is not captured or displayed in the current version of the app.
