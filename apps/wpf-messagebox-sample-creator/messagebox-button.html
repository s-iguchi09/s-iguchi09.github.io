MessageBoxButton — Button Configuration

A ComboBox control for selecting which buttons appear in the MessageBox. This setting controls whether the dialog shows OK, Cancel, Yes, No, Abort, Retry, or Ignore buttons.

← Back to WPF MessageBox Sample Creator

Control: ComboBox

The ComboBox is a standard WPF dropdown control that lets users select one value from a list. In this app, the MessageBoxButton field uses a ComboBox configured with the following properties:

  • IsEditable="False" — The user can only select from predefined values and cannot type a custom value.
  • ItemsSource="{Binding Source={StaticResource MessageBoxButtonDataSource}}" — Automatically populated with all values of the MessageBoxButton enum using ObjectDataProvider.
  • SelectedIndex="1" — Defaults to the second item (OKCancel) when the app starts.
  • SelectedItem="{Binding MessageBoxButton}" — The selected enum value is bound to the ViewModel.

Property: MessageBoxButton

The MessageBoxButton enum is the third argument of MessageBox.Show(). It determines which buttons are displayed in the dialog.

Values and Behavior

OK

Displays a single OK button. The user must click OK to close the dialog. Use this for informational messages that simply need to be acknowledged. Return value: MessageBoxResult.OK.

OKCancel

Displays OK and Cancel buttons. Use this when offering a choice to proceed or cancel an action. Return values: MessageBoxResult.OK or MessageBoxResult.Cancel.

YesNo

Displays Yes and No buttons (no Cancel option). Use this for simple yes/no decisions where deferring or canceling is not allowed. Return values: MessageBoxResult.Yes or MessageBoxResult.No.

YesNoCancel

Displays Yes, No, and Cancel buttons. Use this when the user needs a third option to defer or cancel the decision entirely. Return values: MessageBoxResult.Yes, MessageBoxResult.No, or MessageBoxResult.Cancel.

AbortRetryIgnore

Displays Abort, Retry, and Ignore buttons. Use this for error recovery scenarios where the user can stop, try again, or skip the error. Return values: the MessageBoxResult value corresponding to the button clicked.

RetryCancel

Displays Retry and Cancel buttons. Use this when an operation failed and the user can attempt it again or give up. Return values: the MessageBoxResult value corresponding to the button clicked.

Code Reference

In the app, the selected value is passed as the third argument to MessageBox.Show(): 

MessageBox.Show(
    ViewModel.MessageBoxText,
    ViewModel.Caption,
    ViewModel.MessageBoxButton,  // ← This ComboBox value
    ViewModel.MessageBoxImage,
    ViewModel.MessageBoxResult,
    ViewModel.MessageBoxOptions
);
