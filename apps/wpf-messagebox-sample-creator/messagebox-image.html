MessageBoxImage — Dialog Icon

A ComboBox control for selecting the icon displayed in the MessageBox. The icon visually communicates the nature of the message (error, warning, question, or information).

← Back to WPF MessageBox Sample Creator

Control: ComboBox

The ComboBox is a standard WPF dropdown control that lets users select one value from a list. In this app, the MessageBoxImage field uses a ComboBox with a special data binding approach:

  • ItemsSource="{Binding MessageBoxImagePair}" — Populated from a Dictionary<string, MessageBoxImage> in the ViewModel. This allows multiple enum aliases (e.g., Error, Hand, Stop) to be shown as distinct items with meaningful names.
  • DisplayMemberPath="Key" — The display text shown in the dropdown is the dictionary key (the friendly name string).
  • SelectedValuePath="Value" — The actual bound value is the dictionary value (the MessageBoxImage enum).
  • IsEditable="False" — Selection is limited to the predefined list.
  • SelectedIndex="1" — Defaults to the second item (Error) on startup.

Property: MessageBoxImage

The MessageBoxImage enum is the fourth argument of MessageBox.Show(). It determines the icon shown on the left side of the dialog.

Note: Several enum values map to the same icon. The ViewModel exposes these as separate entries in the dictionary so users can see all available names.

Values and Behavior

None (value: 0)

No icon is displayed. The dialog shows only the message text and buttons without any visual indicator.

Error / Hand / Stop (value: 16)

All three values display the same red circle with a white X icon. Use this for critical error messages or situations requiring immediate attention. The system error sound plays when the dialog is shown.

Question (value: 32)

Displays a blue circle with a white question mark (?) icon. Use this for confirmation dialogs where the user needs to make a decision.

Exclamation / Warning (value: 48)

Both values display the same yellow triangle with a black exclamation mark (!) icon. Use this for warnings about potentially dangerous operations. The system warning sound plays when the dialog is shown.

Asterisk / Information (value: 64)

Both values display the same blue circle with a white lowercase "i" icon. Use this for informational messages that don't require any specific action. The system information sound plays when the dialog is shown.

Code Reference

In the app, the selected value is passed as the fourth argument to MessageBox.Show(): 

MessageBox.Show(
    ViewModel.MessageBoxText,
    ViewModel.Caption,
    ViewModel.MessageBoxButton,
    ViewModel.MessageBoxImage,   // ← This ComboBox value
    ViewModel.MessageBoxResult,
    ViewModel.MessageBoxOptions
);

The ViewModel uses a dictionary to map display names to enum values: 

public Dictionary<string, MessageBoxImage> MessageBoxImagePair { get; set; } = new()
{
    { "None",        MessageBoxImage.None        },
    { "Error",       MessageBoxImage.Error       },
    { "Hand",        MessageBoxImage.Hand        },
    { "Stop",        MessageBoxImage.Stop        },
    { "Question",    MessageBoxImage.Question    },
    { "Exclamation", MessageBoxImage.Exclamation },
    { "Warning",     MessageBoxImage.Warning     },
    { "Asterisk",    MessageBoxImage.Asterisk    },
    { "Information", MessageBoxImage.Information }
};
