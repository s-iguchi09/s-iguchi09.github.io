MessageBoxResult — Default Button

A ComboBox control for selecting which button is focused (highlighted as default) when the MessageBox first appears. This is the button that will be activated if the user presses Enter immediately.

← Back to WPF MessageBox Sample Creator

Control: ComboBox

The ComboBox is a standard WPF dropdown control that lets users select one value from a list. In this app, the MessageBoxResult field uses a ComboBox configured with the following properties:

  • IsEditable="False" — Selection is limited to predefined enum values.
  • ItemsSource="{Binding Source={StaticResource MessageBoxResultDataSource}}" — Automatically populated with all values of the MessageBoxResult enum.
  • SelectedIndex="1" — Defaults to the second item (Cancel) on startup.
  • SelectedItem="{Binding MessageBoxResult}" — The selected enum value is bound to the ViewModel.

Property: defaultResult (MessageBoxResult)

The defaultResult parameter (of type MessageBoxResult) is the fifth argument of MessageBox.Show(). It specifies which button receives focus — and thus acts as the default — when the dialog is displayed. If the user presses Enter without clicking a button, the default button's action is triggered.

Important: The specified default result must correspond to a button that actually exists in the current MessageBoxButton configuration. If the value does not match any available button (for example, specifying Yes when MessageBoxButton.OK is set), the first button in the dialog will receive focus instead.

Values and Behavior

None

No specific default button is set. The system determines which button receives initial focus (typically the first button).

OK

The OK button receives initial focus and is the default action. Applicable when MessageBoxButton is set to OK or OKCancel.

Cancel

The Cancel button receives initial focus and is the default action. Applicable when MessageBoxButton is set to OKCancel, YesNoCancel, or RetryCancel.

Yes

The Yes button receives initial focus and is the default action. Applicable when MessageBoxButton is set to YesNo or YesNoCancel.

No

The No button receives initial focus and is the default action. Applicable when MessageBoxButton is set to YesNo or YesNoCancel.

Code Reference

In the app, the selected value is passed as the fifth argument to MessageBox.Show(): 

MessageBox.Show(
    ViewModel.MessageBoxText,
    ViewModel.Caption,
    ViewModel.MessageBoxButton,
    ViewModel.MessageBoxImage,
    ViewModel.MessageBoxResult,  // ← This ComboBox value (used as defaultResult)
    ViewModel.MessageBoxOptions
);
