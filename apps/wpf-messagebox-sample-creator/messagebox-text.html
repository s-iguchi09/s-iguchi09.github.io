MessageBoxText — Message Body Input

A multi-line TextBox control for entering the body text of the MessageBox. This text appears as the main message content in the dialog.

← Back to WPF MessageBox Sample Creator

Control: TextBox

The TextBox is a standard WPF input control that accepts text from the user. In this app, the MessageBoxText field uses a multi-line TextBox configured with the following properties:

  • AcceptsReturn="True" — Allows the user to press Enter to insert a newline, enabling multi-line message input.
  • VerticalScrollBarVisibility="Auto" — A vertical scrollbar appears automatically when the text exceeds the visible area.
  • HorizontalAlignment="Stretch" — The control stretches to fill the available width.
  • VerticalAlignment="Stretch" — The control stretches to fill the available height of its row.

The Text property is bound to the MessageBoxText property in the ViewModel using UpdateSourceTrigger=PropertyChanged, so the binding updates on every keystroke.

Property: messageBoxText

The messageBoxText parameter is the first argument of MessageBox.Show(). It sets the main body text displayed inside the dialog window.

Behavior by Input

Input Behavior
Empty string The MessageBox displays with no body text. The dialog still shows all configured buttons and icon.
Short text (single line) The MessageBox is compact. The dialog width adjusts to fit the text and button layout.
Long text (single line) The MessageBox auto-resizes horizontally up to a system-defined maximum width, then wraps to multiple lines.
Multi-line text (Enter key used) Line breaks in the input are reflected in the MessageBox, displaying the text across multiple lines.

Code Reference

In the app, the value from this TextBox is passed as the first argument to MessageBox.Show(): 

MessageBox.Show(
    ViewModel.MessageBoxText,   // ← This TextBox value
    ViewModel.Caption,
    ViewModel.MessageBoxButton,
    ViewModel.MessageBoxImage,
    ViewModel.MessageBoxResult,
    ViewModel.MessageBoxOptions
);
